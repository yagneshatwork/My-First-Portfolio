# ✨ Yagnesh Paliwal - Developer Portfolio

Welcome to the source code for my personal developer portfolio! This project is a modern, striking, and fully responsive web application built to showcase my skills, projects, and experiences as a Computer Science student.

## 🎨 Design & Features

The portfolio features a custom **Vibrant Neon & Glassmorphism** aesthetic designed to stand out. 

Key features include:
- **Glassmorphism UI**: Frosted-glass components (`backdrop-blur`) that let the background elegantly shine through.
- **Dynamic Textures & Lighting**: 
  - A subtle CSS film-grain noise overlay providing a premium, tactile feel.
  - Animated, brightly colored, glowing background blobs (`cyan`, `fuchsia`, `purple`) floating behind the content.
- **Micro-Interactions**: Hover states that trigger subtle glowing drop-shadows and geometric transformations.
- **Responsive Layout**: Designed mobile-first to look perfect across all devices.
- **Dark & Light Mode**: Seamless toggling between a warm, vibrant light mode and a deep "Cyber Space" dark mode.
- **Custom Typography**: Utilizing the modern geometric `Outfit` font for a sleek, tech-oriented look.

## 🛠️ Technology Stack

This application is built entirely on the modern frontend stack:
- **[Next.js](https://nextjs.org/)** (v16+) - Framework for React.
- **[Tailwind CSS](https://tailwindcss.com/)** (v4) - Utility-first CSS framework (configured with native CSS variables for custom theming).
- **[Framer Motion](https://www.framer.com/motion/)** - Powerful animation library for all scroll effects and transitions.
- **[Lucide Icons](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)** - Iconography.
- **[Next-Themes](https://github.com/pacocoursey/next-themes)** - Dark/Light mode management.
- **[Sonner](https://sonner.emilkowal.ski/)** - Beautiful toast notifications.

## 🚀 Getting Started Locally

If you'd like to run this project on your own machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yagneshatwork/My-First-Portfolio.git
   cd My-First-Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **View the site:**
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📫 Contact Setup

The contact form is configured using Formspree. If you fork this repository to build your own portfolio, make sure to update the `action` attribute in the `<form>` element inside `app/page.tsx` with your own Formspree URL.
